Customise BootStrap 5 navigation bar

CSS Code

The following CSS code customises the BootStrap 5 navigaton bar. You can view the CSS code within the style tags which can be found in the head of the document.

	.navbar{background-color:red;} // changes background colour of the navigation bar
	.navbar ul li a {color:#fff;} // changes the text color of the active list text links inside the navbar
	.navbar ul li a:hover {color:#fff;} // changes the text color of the hover state text links inside the navbar
	.navbar-brand{color:#00495B;}	// changes the text color of the navbar-brand
	.navbar-nav > li{  padding-left:30px;  padding-right:30px;} // changes the spacing between the text links

Note:

To customise the navigation bar background colour you need to remove the following classes [bg-dark and navbar-dark]:

 <nav class="navbar navbar-expand-sm bg-dark navbar-dark">
